# AI-powered-Interview-Question-Generator
This project is a Python-based app that generates **custom interview questions** using a structured question bank and adaptive logic.  

 Features:
- Dynamic question generation (technical & behavioral)  
- Organized UI with multi-page support (via Streamlit)  
- JSON-based extensible question dataset  
- Ideal for HR, recruiters, and training purposes  

 Structure:
- home.py → main application logic
- pages/ → modular UI pages
- question_bank.json → dataset of questions
